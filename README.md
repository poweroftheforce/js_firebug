# JS Firebug

A JavaScript debugging and inspection utility inspired by early browser developer tools.

## Overview
JS Firebug explores how debugging tools work under the hood and provides a lightweight environment for inspecting JavaScript behavior in real time.

## Features
- JavaScript error logging
- Runtime variable inspection
- Lightweight debugging UI
- Educational tool for understanding JS execution flow

## Tech Stack
- JavaScript (Vanilla JS)
- HTML
- CSS

## What This Project Demonstrates
- Deep JavaScript fundamentals
- Understanding of browser internals
- Debugging and diagnostic thinking
- Custom tooling development

## How to Run
Open the project in a browser or serve it locally using a simple static server.

## Purpose
This project demonstrates my interest in developer tooling and understanding how JavaScript execution behaves at a systems level.
